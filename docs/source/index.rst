星火·链2.0
===================================

BIF-Core2.0（星火·链网底层链2.0,以下简称星火链），是中国信息通信研究院基于区块链技术打造的一条许可公有链，是实现“星火·链网”这一国家级区块链与工业互联网协同创新新型基础设施的关键技术。星火·链网立足安全、自主、可控的原则，面向各行业领域开放，赋能全行业开发者，共建生态繁荣。
